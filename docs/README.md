# Dokumentacja serwera Isle of Berk (Claws of Berk)

Ten folder zawiera szczegółowe instrukcje dotyczące konfiguracji i uruchamiania serwera.

## Pliki
- `INSTALACJA.md`: Instrukcja instalacji serwera Forge i plików modpacka.
- `MODPACK.md`: Szczegóły dotyczące wybranego modpacka (Claws of Berk) i wymaganych modów.
- `HOSTING-SELFHOSTED.md`: Przewodnik po hostowaniu serwera na własnym komputerze (Opcja A).
- `HOSTING-DARMOWY.md`: Porównanie i instrukcje dla darmowych hostingów zewnętrznych (Opcja B).
- `PUBLICZNY-DOSTEP.md`: Informacje o port forwardingu, DNS i bezpieczeństwie.
