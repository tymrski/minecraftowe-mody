# minecraftowe-mody

Isle of Berk (Claws of Berk) - Minecraft HTTYD Server configuration and mod management.

## Project Structure
- `docs/`: Documentation for server setup and hosting.
- `server/`: Server configuration files and run scripts.
- `config/`: Mod lists and other configuration assets.
- `scripts/`: Useful scripts for backups and updates.
- `docker/`: Docker configuration for self-hosting.

## How to use
Refer to `docs/INSTALACJA.md` for setup instructions.
